【C++笔记】友元

发表于 2021-03-08 更新于 2021-08-21 分类于 程序设计 ， C++
有些私有属性，可需要让类外特殊的一些函数或类进行访问，就需要利用友元。友元的关键字为fiend。友元的三种实现方法全局函数做友元、类做友元、成员函数做友元。

有些私有属性，可需要让类外特殊的一些函数或类进行访问，就需要利用友元。

友元的关键字为fiend

友元的三种实现方法

  • 全局函数做友元
  • 类做友元
  • 成员函数做友元

一、全局函数作友元

将全局函数的声明写在类的定义内，并在前面添加关键字friend。

例：friend void func(Person *person);

二、类作友元

语法：friend class className;

三、成员函数作友元

例：friend void className::func();
